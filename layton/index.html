Professor Layton and the
London Holiday
💾 🇯🇵 🗺️ 🇬🇧
London Holiday is a Japan-exclusive
prologue to Diabolical Box 🇺🇸 / Pandora’s
Box 🇪🇺. It is present alongside the Japanese versions of the main game, both in the DS versions (original and Friendly-Ban) and in the HD mobile re-release.

It also could be downloaded as a separate app in the DoCoMo-exclusive Layton Mobile R service, but this version has not yet been preserved.
Inspector Chelmey’s Casebook
~ Special Recipe ~
🇯🇵 🗺️🚧 🇬🇧🚧
Inspector Chelmey’s Casebook ~ Special Recipe is a Japan-exclusive DoCoMo game featuring Chelmey and Barton as protagonists. It contains two parts, which had to be downloaded as two separate apps at a time when the online service was still available (prior to 2014).

A full playthrough has been found by some miracle, making this transcript possible, but this game is still considered lost media due to the fact that no ROM has been retrieved, and that getting your hands on it legally is now impossible.
Professor Layton and the
Mansion of the
Deathly Mirror
🇯🇵🚧 🗺️🚧 🇬🇧🚧
Mansion of the Deathly Mirror is a
Japan-exclusive DoCoMo game with an exclusive story which was about as long as a full game from the main series. It contains six chapters, which had to be downloaded as six separate apps back when the online service was still available (prior to 2014). It is now considered lost media.

This transcript has started but is far from complete, and quite experimental for now. Since we do not have access to the full game, let’s be patient and hope for the best!
Professor Layton and the
Wandering Castle
🇯🇵🚧 🗺️🚧 🇬
Wandering Castle is the first Japan-
exclusive light novel to be released.
Its story is set in the autumn between Diabolical Box 🇺🇸 / Pandora’s Box 🇪🇺 and Unwound Future 🇺🇸 / Lost Future 🇪🇺.
Professor Layton and the
Phantom Deity
🇯🇵🚧 🗺️🚧 🇬
Phantom Deity is the second Japan-
exclusive light novel to be released.
Its story is set when Luke was ten years old, during summer, and spans multiple weeks. It is set after Eternal Diva, which Luke states to have been his first adventure with Professor Layton (aside from the one in Misthallery).
Professor Layton and the
Illusory Forest
🇯🇵🚧 🗺️🚧 🇬
Illusory Forest is the third Japan-
exclusive light novel to be released. It is
set on the same year as Miracle Mask, though it is unclear for now whether it is set before or after it.
Professor Layton and the
Eternal Diva (novel)
🇯🇵 🗺️🚧
The Eternal Diva novel is a Japan-
exclusive novellisation of the movie of
the same name, narrated by both Luke and Janice/Melina. It contains some exclusive scenes that were absent from the movie.
