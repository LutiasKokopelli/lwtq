The game also came out with the following translation files:

  • English
  • Chinese
  • French
  • German
  • Italian
  • Japanese
  • Korean
  • Polish
  • Portuguese
  • Russian
  • Spanish

Translation files contain a copy of the text in English, immediately followed by the same line translated in the given language. There is only one file per language containing every single line in the game, and there is one English translation file.

Surprisingly enough, some lines in the English translation file are not identical to the original line, thus making the original line technically unused in that version of the game.
