Chapter Masterlist

Hello, welcome to Stable Like Sand! As a warning, this fanfic is going to be a long series with a lot of chapters. For example, in the span of just six hours, the beginning of this fanfic is told through a dozen of chapters and more than 60k words! Given the fact that this fanfic’s plot encourages the audience to try to piece things together and solve the mysteries ahead of time, and that a few readers like to go back to earlier chapters in order to look for hints (or just re-read them for fun), I decided to make this index page of sorts in order to make navigation much easier.

If this is your first time around, please ignore this for now! …Or not I guess, I mean, you can do whatever you want. This index will be telling which characters show up in which chapter for example, so if you only want to read a few chapters in which X character shows up, this index will make it entirely possible too. But do know that spoilers are bound to be found here, even though I’m trying to keep them minimal.

Filter Characters:

The following list shows characters who make a physical appearance in the story, in chronological order. If you want to only look for chapters in which that character appears, this section is for you. Just keep in mind that the order in which you press the buttons matters! For example, since multiple characters can appear in the same chapter:

  • The big dark green button will turn off all chapters. If you want to select only chapters in which a certain character appears, then start with this button in order to start fresh and make chapters appear, rather than turn off all characters you don’t care about one by one!
  • Turning on character A will show all chapters in which A appears.
  • However, if afterwards you turn off character B, then all chapters in which B appears will be hidden—even those in which character A also appears.
  • In this case, clicking twice on character A will make these chapters visible once more.
  • Since there is going to be a LOT of characters to sort through, I also added a search bar! If you can’t find them through scrolling, type in the characters you want to toggle on or off.

Fun fact: the search bar will register more than just the displayed name. Characters with titles (e.g. “Professor” or “Dr/Doctor”) or with more than just one name (hello and good day to the dysfunctional Bronev family) will also respond to your queries.
